## Preferences:
Note all the tools listed here may not be used for this project.

### Python
- uv instead of pip
- ruff for python formatting
- typer instead of argparse
- msgspec instead of pydantic
- skops instead of joblib
- use "match-case" instead of "if/elif/elif/elif/else"
- use structlog for logging
- use rich
- use entry points for usualy command
- make tests

### JS
- Vue instead of React/Angular
- biome for formatting

### General
- Git ignore all compiled files (e.g. *.pyc, *.toc, *.log)
- Check tests (fast <1min version) and formatting before allowing commit (add a precommit)
- Open source is generally prefered
- Rust tools are generally prefered
